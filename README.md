# React 开发掘金网站

### 项目介绍

项目技术栈：React.js + react-router + react-redux + redux-thunk + antd + axios + mock.js +  nanoid+ bytemd

### 安装依赖

```js
yarn 
```

### 运行项目

```js
yarn start
```
